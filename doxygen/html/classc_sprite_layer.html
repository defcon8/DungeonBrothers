Detailed Description

Spritelayer: draws a set of sprite's on a SDL Surface.

Optmalisation: When this is True things outside the viewport doesnt get drawed onto the screen surface, this gives a performance boost. Buffered: false: Draw the layer directly onto the main screen on each fRender() call. fRender() should be called each frame update.

true: Render the layer on to the buffer surface when 'once' fRender is called. Then you can retreive the prerendered layer by using fGetBufferSurface().
